# Editorial Flow Quality Bar

Use this checklist when the visual concept is correct but the execution still feels rough.

## 1. Typography

- Title reads immediately from across the canvas
- Subtitle supports, does not compete
- Labels are short enough to be scanned in one breath
- No text sits awkwardly on top of arrows, borders, or decorative shapes
- Line breaks look intentional, not accidental

Red flags:

- title partially covered by blobs or ornaments
- two nearby annotations fighting for the same area
- body copy that turns into texture instead of readable labels

## 2. Composition

- One clear center of gravity
- Edge nodes feel secondary but still readable
- Connectors guide the eye naturally
- Whitespace is preserved around the main anchor
- Legend, if present, sits quietly and does not fight the main story

Red flags:

- eye does not know where to start
- every node has equal weight
- composition looks like a re-labeled dashboard

## 3. Iconography

- Use real-world recognition only when it helps the story
- Prefer a redraw-inspired glyph or a carefully adapted mark over a pasted random icon
- Each icon should be recognizable at small size
- Keep icon style coherent across the visual

Red flags:

- icons feel pasted from different worlds
- one icon is detailed while others are abstract
- icon is there only because the space looked empty

## 4. Tone

- The image should feel like a teaching poster, not a tooling screenshot
- Decorative elements support the mood but do not steal focus
- Sketch energy is welcome; sloppiness is not

Red flags:

- over-clean corporate diagram with no character
- over-styled "hand drawn" effect that hurts readability

## 5. Final Blur Test

Squint or zoom out to ~50%.

You should still be able to tell:

- what the center is
- what the left/right or before/after story is
- where the flow moves
- which nodes matter most

If that fails, the composition is not done.
